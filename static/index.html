WebDB

WebDB is a free, open-source and self-hostable storage method with a simple REST API and no API key! It supports key value storage, secure with passwords and a limit of a thousand characters per variable. We also give one hundred requests per minute to ensure you can easily use this service safely. If you need more features, please email contact@seafoodstudios.com.

You can get started by trying out these Bash commands.

This command will find a variable: 

curl https://webdb.pythonanywhere.com/get/example

This command will create you a variable: 

curl -X POST https://webdb.pythonanywhere.com/create/ \-H "Content-Type: application/json" \-d '{"variable": "example", "password": "securepassword123", "value": "ABC123"}'

This command will delete a variable: 

curl -X POST https://webdb.pythonanywhere.com/delete/ \-H "Content-Type: application/json" \-d '{"variable": "example", "password": "securepassword123"}'
